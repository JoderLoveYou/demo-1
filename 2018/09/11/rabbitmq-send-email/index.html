使用rabbitmq发送qq邮件

2018-09-11
标签： java rabbitmq springboot
类别： 微服务

使用springboot结合rabbitmq来模拟用户注册时，发送电子邮件。

思路

  • 建立两个模块，分别为message_sender和message_receiver；
  • message_sender负责用户注册时，将用户的信息注册到rabbitmq中；
  • message_receiver负责消费消息并发送邮件
